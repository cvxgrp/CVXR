# Direct Standardization: Sample

A sample of [`dspop`](https://www.cvxgrp.org/CVXR/reference/dspop.md)
for direct standardization example. The sample is skewed such that young
males are overrepresented in comparison to the population.

## Usage

``` r
dssamp
```

## Format

A data frame with 100 rows and 3 variables:

- y:

  Response variable

- sex:

  Sex of individual, coded male (0) and female (1)

- age:

  Age of individual
