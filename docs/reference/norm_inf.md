# L-infinity norm of an expression

L-infinity norm of an expression

## Usage

``` r
norm_inf(x, axis = NULL, keepdims = FALSE)
```

## Arguments

- x:

  An Expression

- axis:

  NULL (all), 1 (row-wise), or 2 (column-wise)

- keepdims:

  Logical: keep reduced dimensions?

## Value

A NormInf atom
