# Scalene penalty: alpha \* pos(x) + beta \* neg(x)

Scalene penalty: alpha \* pos(x) + beta \* neg(x)

## Usage

``` r
scalene(x, alpha, beta)
```

## Arguments

- x:

  An Expression

- alpha:

  Coefficient for the positive part

- beta:

  Coefficient for the negative part

## Value

An Expression representing the scalene penalty
