# Set the Value of a Leaf Expression

Assigns a numeric value to a
[`Variable`](https://www.cvxgrp.org/CVXR/reference/Variable.md) or
[`Parameter`](https://www.cvxgrp.org/CVXR/reference/Parameter.md).

## Usage

``` r
value(x) <- value
```

## Arguments

- x:

  A leaf expression object.

- value:

  The value to assign.

## Value

The modified object (invisibly).
