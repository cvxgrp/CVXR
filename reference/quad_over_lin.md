# Sum of squares divided by a scalar

Sum of squares divided by a scalar

## Usage

``` r
quad_over_lin(x, y, axis = NULL, keepdims = FALSE)
```

## Arguments

- x:

  An Expression

- y:

  An Expression (scalar, positive)

- axis:

  NULL (all), 1 (row-wise), or 2 (column-wise)

- keepdims:

  Logical: keep reduced dimensions?

## Value

A QuadOverLin atom
