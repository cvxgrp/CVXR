# Get the Solution Status of a Problem (deprecated)

**\[deprecated\]**

## Usage

``` r
problem_status(x)
```

## Arguments

- x:

  A [`Problem`](https://www.cvxgrp.org/CVXR/reference/Problem.md)
  object.

## Value

Character string, or `NULL` if the problem has not been solved.

Use [`status`](https://www.cvxgrp.org/CVXR/reference/status.md) instead.
