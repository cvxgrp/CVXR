# Log-sum-exp: log(sum(exp(x)))

Log-sum-exp: log(sum(exp(x)))

## Usage

``` r
log_sum_exp(x, axis = NULL, keepdims = FALSE)
```

## Arguments

- x:

  An Expression

- axis:

  NULL (all), 1 (row-wise), or 2 (column-wise)

- keepdims:

  Logical: keep reduced dimensions?

## Value

A LogSumExp atom
