# Sum of squares (= quad_over_lin(x, 1))

Sum of squares (= quad_over_lin(x, 1))

## Usage

``` r
sum_squares(x, axis = NULL, keepdims = FALSE)
```

## Arguments

- x:

  An Expression

- axis:

  NULL (all), 1 (row-wise), or 2 (column-wise)

- keepdims:

  Logical: keep reduced dimensions?

## Value

A QuadOverLin atom
