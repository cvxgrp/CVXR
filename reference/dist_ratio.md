# Distance ratio

Computes norm(x - a)\_2 / norm(x - b)\_2, where a and b are constants.
This is a quasiconvex atom.

## Usage

``` r
dist_ratio(x, a, b)
```

## Arguments

- x:

  A vector expression

- a:

  A numeric constant vector

- b:

  A numeric constant vector

## Value

An expression representing the distance ratio
