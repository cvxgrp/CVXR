# Invert a Solution through a Reduction

Invert a Solution through a Reduction

## Usage

``` r
reduction_invert(x, solution, inverse_data, ...)
```

## Arguments

- x:

  A Reduction object.

- solution:

  A solution object.

- inverse_data:

  Inverse data from apply.

- ...:

  Additional arguments.

## Value

A solution to the original problem.
