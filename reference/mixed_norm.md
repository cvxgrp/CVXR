# Mixed norm (\\L\_{p,q}\\ norm): column-wise p-norm, then q-norm

Mixed norm (\\L\_{p,q}\\ norm): column-wise p-norm, then q-norm

## Usage

``` r
mixed_norm(X, p = 2, q = 1)
```

## Arguments

- X:

  An Expression (matrix)

- p:

  Inner norm parameter (default 2)

- q:

  Outer norm parameter (default 1)

## Value

An Expression representing the mixed norm
