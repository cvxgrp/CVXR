# Get the Raw Solution Object (deprecated)

**\[deprecated\]**

## Usage

``` r
problem_solution(x)
```

## Arguments

- x:

  A [`Problem`](https://www.cvxgrp.org/CVXR/reference/Problem.md)
  object.

## Value

A `Solution` object, or `NULL` if the problem has not been solved.

## Details

Use [`solution`](https://www.cvxgrp.org/CVXR/reference/solution.md)
instead.
